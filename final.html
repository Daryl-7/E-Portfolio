// Overall Reflection

What I Learned This Semester

Throughout this semester in Object-Oriented Programming, I was able to build a solid foundation in Java and deepen my understanding of core OOP concepts. Starting from the basics — variables, data types, and operators — all the way to creating full programs like an ATM System, an Enrollment Assessment System, and an Expense Tracker, each activity pushed me to apply what I learned in a practical way. Some activities came naturally to me, especially the ATM System since I had prior experience with something similar, while others like the Enrollment Assessment System challenged me due to the number of conditions involved. My favorite output was the Expense Tracker, where I got to showcase my knowledge of void and non-void methods. Overall, this course helped me grow not just as a programmer, but in how I approach and solve problems — and getting 40 out of 45 on the midterm quiz is a reflection of that growth.

// How to enable downloads on each card

  1. Save your file (e.g. Activity1.java) in the same folder as this HTML file.
  2. Find the card's <a> download button and change href="#" to href="Activity1.java".
  3. Add download="Activity1.java" as an attribute so it triggers a download instead of opening.
  4. Remove class="disabled" from the button so it turns green and becomes clickable.
  5. For the preview image, save a screenshot as e.g. activity1.jpg and replace the placeholder div with <img src="activity1.jpg" />.
  6. Update the file-badge class to match the file type: badge-java, badge-img, badge-pdf, or badge-mixed.
